# Urop
A simple URL-Shorter based on Node.js and TypeScript.

## Requirement
<ul>
<li>Node.js</li>
<li><del>NPM (will be installed with nodejs)</del></li>
<li>TypeScript</li>
</ul>

## Packages
```
npm install @types/node ts-node express @types/express quick.db body-parser
```

## Run
```
tsc && app.js
```
